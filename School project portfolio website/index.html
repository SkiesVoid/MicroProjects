GitHub Projects

Below are several projects I have created and published on GitHub. These projects focus on practical tools, networking experiments, and software utilities built primarily with Python and the google API.

  • GSDBE.py – Google Sheets Database Editor V0.2.0
    A Python tool that allows Google Sheets to be used as a lightweight database, supporting structured data editing and management through the Google Sheets API.
  • LevisInternetSpeedDetector.py – Network Speed and Packet Detector
    A networking utility that measures internet speed and analyzes packet activity to help diagnose network performance issues.
  • LocalVoIPApp.py – Local Network LAN VoIP App
    A work-in-progress voice-over-IP application designed for local area networks, enabling voice communication without external servers.
  • PasswordGenerator.py – Simple 12 Character Password Generator
    A Python script that generates secure, randomized 12-character passwords for basic security and credential use cases.
  • System Process Scanner.exe – System Process Scanner
    A compiled utility that scans and lists running system processes, designed to help monitor and analyze system activity.
  • Void Browser Version 0.1.2.0.exe – Custom Web Browser
    A fully independent web browser built from scratch without relying on existing browser frameworks, focusing on custom rendering and control logic.
  • privateinternalIMS.py – Private LAN Instant Messaging System
    A work-in-progress application that uses local area networking to create one-time, private instant messaging sessions between two computers.
